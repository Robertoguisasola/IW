Para acceder a la aplicación, ejecutar el comando python manage.py runserver dentro de la carpeta Ski Stick
Una vez realizado esto, escribir en la barra de busqueda del navegador http://127.0.0.1:8000/index
En caso de querer acceder a la parte del administrador, escribir http://127.0.0.1:8000/admin
El administrador general de la aplicacion es admin, y de contraseña admin
También hemos creado un administrador aparte con menos derechos, cuyo nombre de usuario es ikerpastor y su contraseña iker1234
